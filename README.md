This interactive piano widget lets you play music on your XENEON EDGE! It prioritizes maximum performance and low-latency response using real-time Web Audio synthesis. This approach eliminates file-loading overhead for a lightweight, zero-load implementation. You can continue to play music without an internet connection!

Note: AI was used as a tool to assist in the development of this widget, however, I made several manual adjustments to the code to ensure its functionality and the final desired outcome.
